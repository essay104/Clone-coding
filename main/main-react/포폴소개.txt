리액트 사용

리액트를 사용한 이유?
리액트가 가지고 있는 컴포넌트를 재활용하는 기능과 원페이지 구성 방식에 매력을 느꼈기 때문

npm
스타일컴포넌트 사용
프레임 모션 사용
풀페이지 사용(프로젝트페이지)


1 메인화면
1) 다크모드-라이트모드 전환 구현
2) 현재위치의 위도와 경도값을 구하고 OpenWhter API를 통해 현재 지역의 날씨, 온도를 헤더에 출력
3) new Date()로 현재 날짜를 헤더에 출력하고 useEffect와 setInterVal로 현재 시간을 갱신하도록 구현
4) Home About Project같은 nav menu들을 클릭하여 페이지를 전환할 수 있게 만들었음 (useNavigate와 Link를 사용)
5) 하단에 페이저를 구현하여 사용자가 현재 페이지가 전체 문서에 어느부분에 위치하고 있는지 확인할 수 있게 만들었으며
해당 페이저를 클릭하여 화면을 이동할 수 있도록 만들었음
6) 페이저는 화살표와 nav menu들과 연동될 수 있도록 만들었으며 이를 구현하기 위해 페이저 함수를 useContext로 ReactTree 전역에서 사용할 수 있도록 만들었음

2 ABOUT화면
1) useState로 각 섹션을 클릭할때 flex값을 true로 줘서 삼항조건 연산자로 flex값이 바뀌게 만들었음
2) skillsection은 클릭시 다른 페이지로 넘어가게 했고 x버튼을 눌러 이전페이지로 돌아가게 만들었음

3 Project화면
1) 풀페이지 방식으로 사용자가 유연하게 하단으로 이동할 수 있게 만들었음
